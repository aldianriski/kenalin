How this page embeds Kenalin

One script tag with data-api-url is enough — it registers the <kenalin-ai> Web Component and auto-mounts it:

<script src="/kenalin.js" data-api-url="http://localhost:8787" defer></script>

Or place the element yourself to control position / page context:

<kenalin-ai api-url="http://localhost:8787" project-id="quickhub"></kenalin-ai>
