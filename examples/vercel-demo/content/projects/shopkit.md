# ShopKit

**ShopKit** is an open-source e-commerce starter (a demo project) that gets a
small shop online without a heavyweight platform. It bundles a product catalog,
a cart, and a checkout flow behind a clean, themeable storefront.

Alex built ShopKit to be readable and easy to extend: a typed API, server-side
rendering for fast first loads and SEO, and a payment integration that's a single
adapter to swap. Store owners can restyle the whole front end from a small set of
design tokens.

**Stack:** Next.js, TypeScript, Stripe-style payment adapter, Postgres.
**Outcome:** a starter teams can fork and launch a store from in a day.
