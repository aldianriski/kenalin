# TaskFlow

**TaskFlow** is a lightweight task manager for small teams — a demo project that
shows how Alex approaches product work. The goal was a tool a five-person team
could adopt in an afternoon: boards, assignees, and due dates, with none of the
enterprise clutter.

Alex designed the data model, built the React front end and a Node + Postgres
API, and shipped a real-time board that updates as teammates move cards. The
result was a focused app that stayed fast as projects grew, with keyboard-first
navigation and a clean, calm interface.

**Stack:** React, TypeScript, Node, Postgres, WebSockets.
**Outcome:** a maintainable codebase and a UI the team actually enjoyed using.
